## Accepted Tech Debt

- Remote and sandbox targets are integrated as durable placement metadata plus explicit local-runtime exclusion, not as a built-in remote transport or provisioning system. This is consistent with Phase 08 as shipped, but it means non-local workers still depend on operator-managed execution and shared state access.
- Timeline lifecycle coverage for non-local targets depends on those targets writing the same durable `state.jsonl` events consumed by `internal/session/run_timeline.go:273-340`. Without that, routing/blocker/replan events remain visible, but accept/wait/block/done transitions will be incomplete.
- Adaptive preferences refresh only when the coordinator root message is completed in `cmd/tmuxicate/main.go:943-959`. Runs abandoned before root completion will not feed future adaptive decisions until explicitly closed.
